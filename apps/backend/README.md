# Lex Love Loom вЂ” Backend (NestJS)

Hybrid NestJS API layered over Lovable Cloud (Supabase). See [`/docs/MIGRATION_PLAN.md`](../../docs/MIGRATION_PLAN.md).

## Stack

- **NestJS 10** вЂ” modular, DI, decorators
- **Zod** вЂ” runtime validation via custom pipe
- **Swagger** вЂ” autogenerated API docs at `/docs`
- **Pino** вЂ” structured JSON logging
- **Supabase JS** вЂ” DB / Auth verification (JWKS)
- **Throttler** вЂ” rate limiting

## Quick start

```bash
cd apps/backend
cp .env.example .env   # fill SUPABASE_* + LOVABLE_API_KEY
npm install
npm run start:dev      # http://localhost:3001
```

- Swagger: http://localhost:3001/docs
- Health: http://localhost:3001/health

## Auth

All routes (except `/health` and `/docs`) require `Authorization: Bearer <supabase_jwt>`.
The JWT is verified against Supabase's JWKS endpoint, claims attached to `req.user`.
Role checks use `@Roles('admin')` decorator backed by `public.has_role()`.
